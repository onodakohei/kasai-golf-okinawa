# 沖縄ゴルフ会員権ランディングページ

カサイゴルフが取り扱う沖縄のゴルフ場の会員権のランディングページです。

## ファイル構成

- `index.html` - メインのHTMLファイル
- `assets/styles.css` - スタイルシート
- `assets/script.js` - JavaScriptファイル
- `assets/images/` - 画像ファイル（hero.jpg, logo.png）

## GitHub Pagesでの公開方法

1. GitHubで新しいリポジトリを作成（Privateリポジトリ推奨）
2. このリポジトリをGitHubにプッシュ
3. リポジトリのSettings > Pagesで、Sourceを「main」ブランチに設定
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]/` でアクセス可能になります

## 限定公開について

- Privateリポジトリにすることで、リポジトリへのアクセスを制限できます
- GitHub PagesはPrivateリポジトリでも利用可能です（無料プランでは制限あり）
- より厳密な限定公開が必要な場合は、GitHubのCollaborators機能で特定のユーザーのみアクセス可能に設定できます
